Credenciales del proyecto donde se guardan tokens, torneos y demás datos.

URL del proyecto

Streaming
Kick

Credenciales OAuth, canal y autorización del bot.

Credenciales OAuth
Client ID y Secret de tu aplicación en Kick Developers.
Canal de Kick (sin @)
Nombre del canal donde corre el bot. Si autorizás OAuth, se completa solo.
Chatroom ID
ID numérico del chatroom del canal.
Para obtenerlo: abrí en el navegador https://kick.com/api/v2/channels/tu_canal y copiá chatroom.id.
ID de reward para song request
Este ID viene de Supabase por ambiente. Para cambiarlo, usá la gestión de rewards desde la pestaña Kick.
